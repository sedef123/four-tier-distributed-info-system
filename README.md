# Four-Tier Distributed Information System

This is a comprehensive full-stack distributed system built as an academic project for the **SWE212** course. It features a modern four-tier architecture separating the database, business logic, view controller, and user client.

## 🛠️ Architecture & Technologies
* **Tier 1 (Database):** MySQL / PostgreSQL
* **Tier 2 (Back-end):** Spring Boot (v4.0.1) REST API running on JVM
* **Tier 3 (Front-end):** Node.js (v24.13.0) & React / Vite (v8.2.0) ViewAPI
* **Tier 4 (Client):** Modern Web Browser

## ✨ Key Features
* **Distributed Architecture:** Decoupled front-end and back-end servers communicating via endpoints.
* **Image Management:** Full support for secure image upload and download operations.
* **Dynamic PDF Reporting:** Integrated with **JasperStudio** community edition to generate structured tabular reports directly from database entities.
* **API Testing:** Thoroughly tested endpoints using Postman.
